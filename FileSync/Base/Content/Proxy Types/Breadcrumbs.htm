Breadcrumbs Proxy

The Breadcrumbs Proxy is used as a placeholder for a breadcrumbs path in HTML output. It is generally inserted into a MasterPage.

Viewing the Breadcrumbs Proxy in this project source

Content Explorer > MasterPages > default.flmsp

In the MasterPage, it is labeled breadcrumps proxy.

How the Breadcrumbs Proxy works in this project

The default.flmsp MasterPage is assigned to the HTML5 target (see the Advanced tab of the target). When you build the HTML5 target, the default.flsmp MasterPage is applied to each topic, and the breadcrumbs path is generated for each topic in the output.

Try it! Inserting a Body Proxy

Create a new MasterPage by right clicking Content Explorer > Resources > MasterPages and selecting New > MasterPage, or use an existing MasterPage. Once the new MasterPage is in place, click Insert > Proxy > Breadcrumbs Proxy in the MasterPage.
