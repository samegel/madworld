Concepts Proxy

The Concepts Proxy is used to generate a list of Concepts used in your Flare project. It is inserted into a topic. A Concept is a specific item in Flare used for a number of purposes.Learn more about Concepts at http://help.madcapsoftware.com/flare2017/Content/Concepts/About_Concepts.htm

Viewing the Concepts Proxy in this project source

Content Explorer > List of Concepts.htm

How the Concepts Proxy works in this project

Each topic in this project contains at least one Concept entry, indicated by the green box at the top of the topic. Click View > Concept Window to see all the Concepts used in the project.

Concepts Proxy Dependencies

Concept entries in the topics.

Try it! Inserting a Concept Proxy

Create a new topic, and then click Insert > Proxy > Insert Concepts Proxy into a paragraph in the new topic.
