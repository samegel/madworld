Endnotes Proxy

The Endnotes Proxy is used to consolidate footnotes at the end of a manual. It is generally inserted into a topic and conditioned for print only.

Viewing the Endnotes Proxy in this project source

Content Explorer > Endnotes.htm

How the Endnotes Proxy works in this project

In some of the topics, there are footnotes (inserted by clicking Insert > Footnote). When you build a target, the footnotes are consolidated into the endnotes proxy in the Endnotes.htm topic.Learn more about Endnotes and Footnotes at http://help.madcapsoftware.com/flare2017/Content/Footnotes/About_Footnotes.htm

Endnotes Proxy Dependencies

Footnotes inserted into topics, with the position of the footnote set to Endnotes proxy.

Try it! Inserting an Endnotes Proxy

Create a new topic. In that topic, select Insert > Proxy > Insert Endnotes proxy. Then, pick any topic, and add a footnotes by clicking Insert > Footnote and selecting Endnotes proxy for the footnote location.
