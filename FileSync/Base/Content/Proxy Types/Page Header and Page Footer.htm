Page Header & Page Footer

The Page Header and Page Footer proxies generate running headers and footers in Word and FrameMaker output. They are inserted into a print MasterPage. That MasterPage is then linked to the Word target.

Viewing the Page Header and Footer Proxies in this project source

Content Explorer > Resources > MasterPages > Word.flmsp

How the Page Header and Footer Proxies work in this project

The Word.flmsp MasterPage contains both a Page Header and Page Footer proxy. The Header proxy prints the most recent H1, and the Footer proxy prints the page number. This MasterPage is then specified in the Advanced section of the Word target.

Page Header and Footer Proxy Dependencies

The page header relies on the Running HF variable file. The Running H/F 1 variable is used in the Page Header proxy.

Try it! Inserting an Index Proxy

Create a new MasterPage. In that topic, select Insert > Page Header or Page Footer.
