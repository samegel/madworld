Glossary Proxy

The Glossary Proxy is used to generate a glossary in your Flare output. It is inserted into a topic.

Viewing the Glossary Proxy in this project source

Content Explorer > Glossary.htm

How the Glossary Proxy works in this project

The Glossary proxy builds the contents of the Project Organizer > Glossaries > Glossary file in targets where this glossary file is selected (see the Glossary tab in the target files). You can have more than one glossary. Each glossary displays in the target file, and you can select which glossaries apply to that target.Learn more about Glossaries at http://help.madcapsoftware.com/flare2017/Content/Glossaries/About_Glossaries.htm

Glossary Proxy Dependencies

Glossary file or files in the project, and glossary files selected in one or more targets.

Try it! Inserting a Glossary Proxy

Create a new topic. In that topic, select Insert > Proxy > Insert Glossary proxy.
