Index Proxy

The Index Proxy is used to generate an index. It is inserted into a topic.

Viewing the Index Proxy in this project source

Content Explorer > Index.htm

How the Index Proxy works in this project

Each topic in this project has index terms associated with it, indicated by the light green boxes after the H1. You can view all index entries by clicking View > Index Window. When the target is built, Flare collects all the index entries and replaces the index proxy with the index.Learn more about indexes at http://help.madcapsoftware.com/flare2017/Content/Indexes/About_Indexes.htm
