Relationships Proxy

The relationships proxy is an element you can place in a topic or MasterPage to automatically create links between related topics. These links are based on a relationships table that you create. If a topic is inserted into the relationship table, a Relationships Proxy is in the same topic, and the Relationships Table is included in a target, the a list of related topics will automatically be generated based on the relationships table.

Viewing the Relationships Proxy in this project source

Content Explorer > Relationships.htm (bottom of this topic)

How the Relationships Proxy works in this project

There is a relationship table in Project Organizer > Advanced > Relationship Tables folder. That table includes this topic as well as other topics that are related to it. This relationship table is then selected in the Relationship Table tab of the target. Finally, the relationships proxy is inserted at the bottom of this topic.

For information about creating relationship tables, see http://help.madcapsoftware.com/flare2017/Content/Nav_Links/Relationship_Links/Adding_Relationship_Tables.htm.
