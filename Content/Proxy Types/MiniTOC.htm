Mini-TOC Proxy

The Mini-TOC Proxy is used to generate a portion of your TOC or topic headings at a particular location in the output. Mini-TOC proxies are usually placed in topics, although it's not uncommon for them to be in MasterPages.

Viewing the Mini-TOC Proxy in this project source

Content Explorer > In This Guide.htm

How the MiniTOC Proxy works in this project

The Mini-TOC is inserted into a topic where all the proxy types are listed as children of that topic (see the Master TOC file). As such, each of the children to the In This Guide.htm topic will appear where the Mini-TOC proxy is placed.

If you want a Mini-TOC to appear in each of your topics for online output, then place a Mini-TOC proxy in the MasterPage.
