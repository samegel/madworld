List-Of Proxy

The List-Of Proxy is used to generate a list of any element / class name combination used in your project.

Viewing the Index Proxy in this project source

Content Explorer > List-Of Proxy.htm

How the List-Of Proxy works in this project

In the List-Of Proxy.htm topic, there are two List-Of proxies. One lists all instances of <span class="PathName"> and the other lists all instances of <h2>. Generally, List-Of proxies are used to create lists of tables or images.Learn more at http://help.madcapsoftware.com/flare2017/Content/Proxies/Creating_a_List_of_Elements.htm

Try it! Inserting a List-Of Proxy

Create a new topic. In that topic, select Insert > Proxy > Insert List-Of proxy. In the List-Of proxy dialog box, select an element and class name from the drop-down lists.
